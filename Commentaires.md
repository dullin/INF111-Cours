Deux types: 

* Ligne simple débutant par `//`
* Plusieurs lignes délimité par `/*` et `*/`


```java
/*
Une commentaire
sur plusieurs lignes.
*/
public class HelloWorld{
    public static void main(String[] args){
        // Affiche un message.
        System.out.println("Hello, world!"); //Marche ici aussi!
    }
}
```
